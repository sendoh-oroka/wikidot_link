# SCP-JP ガイドページ リンク可視化

SCP-JPのガイドページ間のリンク関係をD3.jsで可視化するツール。

## 可視化ページの見方・操作方法

### グラフの見方

| 要素 | 意味 |
|------|------|
| ノードの大きさ | 被リンク数（他のページからリンクされているほど大きい） |
| 矢印の向き | リンク元 → リンク先 |

### ノードの色分け

| 色 | 意味 |
|----|------|
| 白〜水色（デフォルト） | 通常のノード |
| 黄色 | 選択中のノード |
| 緑 | 選択ノードがリンクしている先（リンク先） |
| オレンジ | 選択ノードにリンクしている元（被リンク元） |
| 暗色 | 経年フィルターで対象外のノード |

### 操作方法

- **ノードをクリック** — 詳細パネルを開き、リンク先・被リンク元の一覧と関連ノードを強調表示
- **Shift+クリック** — 複数のノードを同時に選択
- **背景をクリック** — 選択を解除
- **マウスホイール** — ズームイン・アウト
- **背景ドラッグ** — パン（グラフ全体を移動）
- **ノードドラッグ** — ノードを任意の位置に移動
- **詳細パネルのリンク一覧をクリック** — 対応するノードを選択

### 詳細パネルに表示される情報

- タイトル
- URL（クリックでWikidot上の該当ページへ遷移）
- 著者（クリックでWikidotユーザーページへ遷移）
- 作成年
- リンク先一覧
- 被リンク元一覧

## UIコントロールの使い方

### 経年フィルター（年を入力）

指定した年以前に作成されたノードが強調表示されます。それより新しいノードは暗くなります。空欄にすると全ノードが同じ明るさで表示されます。

例: `2020` と入力すると、2020年以前に作成されたページが明るく表示される

### 相互リンクフィルター

押すとA→B かつ B→A の双方向リンクを持つノード同士のエッジのみ表示されます。

### SCP財団除外フィルター

押すと著者がSCP財団でないノードのみ表示されます。

### 検索ボックス

テキストを入力すると、タイトルまたはURLに一致するノードが強調表示されます。

## ローカルでの確認方法

`data.json` をJavaScriptから読み込むため、`file://` プロトコルでは動作しません。ローカル確認にはHTTPサーバーが必要です。

```sh
# リポジトリのルートディレクトリで実行
python -m http.server 8000
# → ブラウザで http://localhost:8000 を開く
```

## ftmlファイル追加・更新手順

新しいガイドページを追加したり、既存ページのデータを更新する場合の手順です。

```sh
# 1. ftml/ ディレクトリに新しいftmlファイルを追加
#    ファイル名は file-list.csv の url フィールドに合わせる（例: new-page.ftml）

# 2. file-list.csv に新しいエントリを追加（末尾の # currentDate 行より前に追加する）
#    フォーマット: url,title,author,createdAt
#    例:
#    new-page,新しいガイド,SCP財団,2025/01/01

# 3. data.json を再生成
uv run extract_links.py

# 4. コミット・プッシュ
git add data.json ftml/<新ファイル名>.ftml file-list.csv
git commit -m "feat: add <ページ名>"
git push
```

## GitHub Pages デプロイ手順

1. GitHubにリポジトリをpushする
2. リポジトリの **Settings** → **Pages** を開く
3. **Source** を `Deploy from a branch` に設定
4. ブランチを `main`、フォルダを `/ (root)` に設定して **Save**
5. しばらく待つと `https://<username>.github.io/<repo-name>/` でアクセス可能になる

## file-list.csv のフォーマット

```csv
url,title,author,createdAt
site-rules,サイトルール,SCP財団,2024/08/18
```

| フィールド | 説明 |
|-----------|------|
| `url` | Wikidot上のページパス（`http://scp-jp.wikidot.com/` 以降の部分） |
| `title` | ページタイトル |
| `author` | 著者名。合作の場合は `{author1,author2}` 形式 |
| `createdAt` | 作成日（`YYYY/M/D` 形式） |

著者が `SCP財団` の場合、詳細パネルの著者リンクは表示されません。

```
    ├── site-rules.ftml
    └── ...
```
